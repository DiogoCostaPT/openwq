Overview

OpenWQ can be coupled to existing hydro-models through the series of generic coupler functions. These coupler functions have three main purposes:

  • Convert datatypes between the hydro-model and OpenWQ

  • Call OpenWQ’s APIs

The main OpenWQ APIs are:

  • OpenWQ::decl (see API reference)

  • OpenWQ::run_time_start (see API reference)

  • OpenWQ::run_space (see API reference)

  • OpenWQ::run_time_end (see API reference)
