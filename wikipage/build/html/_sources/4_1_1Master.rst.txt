Master
==================================

The master configuration is a JSON file that provides OpenWQ with information and instructions to run a simulation. It contains key-values pairs defined below: 

+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| Key -> Sub-key(s)                                                         | Description                                                                                                   |
+===========================================================================+===============================================================================================================+
| ``PROJECT_NAME``                                                          | Project name string                                                                                           |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``GEOGRAPHICAL_LOCATION``                                                 | Project location string                                                                                       |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``AUTHORS``                                                               | Model run authors                                                                                             |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``DATE``                                                                  | Model run data                                                                                                |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``COMMENT``                                                               | Additional documentation information                                                                          |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``COMPUTATIONAL_SETTINGS``                                                |                                                                                                               |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``COMPUTATIONAL_SETTINGS`` -> ``RUN_MODE_DEBUG``                          | Run model debugging flag (true/false)                                                                         |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``COMPUTATIONAL_SETTINGS`` -> ``USE_NUM_THREADS``                         | "all" or integer of number of threads to use (e.g., 10)                                                       |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``OPENWQ_INPUT``                                                          |                                                                                                               |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``OPENWQ_INPUT`` -> ``CONFIG_FILEPATH``                                   | `Main configuration file path <https://openwq.readthedocs.io/en/latest/4_1_2Config.html>`_                    |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``OPENWQ_INPUT`` -> ``EXTERNAL_WATER_FLUXES``                             | Numbered pairs of LABEL and FILEPATH for external-fluxes input files                                          |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``OPENWQ_INPUT`` -> ``EXTERNAL_WATER_FLUXES`` -> ``(#)`` -> ``LABEL``     | Label id for input file                                                                                       |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``OPENWQ_INPUT`` -> ``EXTERNAL_WATER_FLUXES`` -> ``(#)`` -> ``FILEPATH``  | File path to `External Water Fluxes file <https://openwq.readthedocs.io/en/latest/4_1_4Source.html>`_         |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``OPENWQ_INPUT`` -> ``SINK_SOURCE``                                       | Numbered pairs of LABEL and FILEPATH for chemical-sources input files                                         |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``OPENWQ_INPUT`` -> ``SINK_SOURCE``           -> ``(#)`` -> ``LABEL``     | Label id for input file                                                                                       |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``OPENWQ_INPUT`` -> ``EXTERNAL_WATER_FLUXES`` -> ``(#)`` -> ``FILEPATH``  | Path to `Sink and Source file <https://openwq.readthedocs.io/en/latest/4_1_4Source.html>`_                    |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``MODULES``                                                               |                                                                                                               |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``MODULES`` -> ``BIOGEOCHEMISTRY``                                        | Configuration of Biogeochemical reactions                                                                     |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``MODULES`` -> ``BIOGEOCHEMISTRY`` -> ``MODULE_NAME``                     | Biogeochemical cycling module choice                                                                          |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``MODULES`` -> ``BIOGEOCHEMISTRY`` -> ``MODULE_CONFIG_FILEPATH``          | Path to `Biogeochemical cycling configuration file <https://openwq.readthedocs.io/en/latest/4_1_3BGC.html#>`_ |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``MODULES`` -> ``TRANSPORT_EROSION``                                      | Configuration of Transport and Erosion                                                                        |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``MODULES`` -> ``TRANSPORT_EROSION`` -> ``MODULE_NAME``                   | Transport-Erosion module choice                                                                               |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``MODULES`` -> ``TRANSPORT_EROSION`` -> ``MODULE_CONFIG_FILEPATH``        | Transport-Erosion configuration file path                                                                     |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``OPENWQ_OUTPUT``                                                         | Output configuration                                                                                          |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``OPENWQ_OUTPUT`` -> ``RESULTS_FOLDERPATH``                               | Output folder path                                                                                            |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``OPENWQ_OUTPUT`` -> ``FORMAT``                                           | Output file format ( `VTU, CSV, or HDF5 <https://openwq.readthedocs.io/en/latest/4_2_1file.html>`_)           |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``OPENWQ_OUTPUT`` -> ``CHEMICAL_SPECIES``                                 | Chemical species to include in output files                                                                   |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``OPENWQ_OUTPUT`` -> ``UNITS``                                            | Units of outputs of chemical species                                                                          |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``OPENWQ_OUTPUT`` -> ``NO_WATER_CONC_FLAG``                               | Default value for concentration when there is not water                                                       |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``OPENWQ_OUTPUT`` -> ``COMPARTMENTS_AND_CELLS``                           | Names of compartments and indices to output                                                                   |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| ``OPENWQ_OUTPUT`` -> ``TIMESTEP``                                         | Temporal resolution of output file e.g., ``[1,"hour"]``                                                       |
+---------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+

Example:

.. code-block:: json

    {
        "PROJECT_NAME": "My_Project",
        "GEOGRAPHICAL_LOCATION": "Saskatoon",
        "AUTHORS": "Diogo Costa",
        "DATE": "Aug_2022",
        "COMMENT": "This is my first project.",
        "COMPUTATIONAL_SETTINGS": {
            "RUN_MODE_DEBUG": true,
            "USE_NUM_THREADS": "all"
        },
        "OPENWQ_INPUT": {
            "CONFIG_FILEPATH": "openWQ_config.json",
            "EXTERNAL_WATER_FLUXES": {
                "1": {
                    "LABEL": "Rainfall_example",
                    "FILEPATH": "openWQ_rainfall.json"
                }
            },
            "SINK_SOURCE": {
                "1": {
                    "LABEL": "External_Load_example",
                    "FILEPATH": "openWQ_source_batchSimulations.json"
                }
            }
        },
        "MODULES": {
            "BIOGEOCHEMISTRY": {
                "MODULE_NAME": "OPENWQ_NATIVE_BGC",
                "MODULE_CONFIG_FILEPATH": "openWQ_MODULE_BGC_cycling.json"
            },
            "TRANSPORT_EROSION": {
                "MODULE_NAME": "OPENWQ_NATIVE_TE_NO_ADVDISP"
                "MODULE_CONFIG_FILEPATH": "openWQ_MODULE_TE_batchSimulations.json"
            }
        },
        "OPENWQ_OUTPUT": {
            "RESULTS_FOLDERPATH": "Output_OpenWQ",
            "FORMAT": "HDF5",
            "CHEMICAL_SPECIES": ["species_A", "species_B"],
            "UNITS": "kg",
            "NO_WATER_CONC_FLAG": -9999,
            "COMPARTMENTS_AND_CELLS": {
                "SCALARAQUIFER": {
                    "1": [1, 1, 1]
                }
            },
            "TIMESTEP": [1, "day"]
        }
    }
