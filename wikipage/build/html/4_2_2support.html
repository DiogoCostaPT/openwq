Read outputs

The different output file formats enabled in OpenWQ are all fairly well established in the scientific community: CSV, HDF5, and VTU. Widely used programming languages such as C++, R, Python and MATLAB have support for reading, editing and creating such files.

However, we developed a suite of supporting scripts specifically tailored to read OpenWQ’s outputs. Check them here.
